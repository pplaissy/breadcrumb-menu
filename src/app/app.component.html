Primeng Menubar Primeng Breadcrumb

Each segment of the breadcrumb is a menu level. When an item is selected, an additional level is added to the breadcrumb with the children of the selected item, and this level opens automatically to smooth the selection. The cycle ends when the last item selected has no children. You can return to any level of the breadcrumb and follow a different path. If the breadcrumb loses focus before the selection cycle is complete, the first item is selected by default.
